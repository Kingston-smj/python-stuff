Certainly! Here's a Python program that prompts for a file name, reads through the file, and extracts the spam confidence values to calculate the average spam confidence:

python
def extract_spam_confidence(file_name):
    try:
        file = open(file_name, 'r')
    except FileNotFoundError:
        print("File not found.")
        return

    count = 0
    total_spam_confidence = 0

    for line in file:
        if line.startswith("X-DSPAM-Confidence:"):
            # Extract the spam confidence value from the line
            spam_confidence = float(line.split(':')[1])
            count += 1
            total_spam_confidence += spam_confidence

    file.close()

    if count > 0:
        average_spam_confidence = total_spam_confidence / count
        print("Average spam confidence:", average_spam_confidence)
    else:
        print("No lines with spam confidence found.")

file_name = input("Enter the file name: ")
extract_spam_confidence(file_name)

You can run this program and enter the file name (e.g., mbox.txt or mbox-short.txt) when prompted. The program will then read the file, extract the spam confidence values, and calculate the average spam confidence.
